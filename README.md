 Project Overview
This project focuses on manual testing of an E-Commerce checkout flow to validate the end-to-end user journey. The goal was to ensure that each step of the checkout process works correctly, from adding items to the cart through successful order confirmation.
Although the task description mentions automation, this project was executed using manual testing techniques to analyze UI behavior, form validation, and page transitions.
